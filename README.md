# USB-PDO-Decoding-Helper

A small excel file for decoding USP PowerDelivery PDO's

I created it to verify my PDO-Decoder form this Library:
https://github.com/cad435/CH224Q_Arduino

Fill In your PDO-Values as 4xBytes and the sheet will calculate all values for fixed, battery, variable and augmented fields.
